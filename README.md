# Unit Testing (Pytest)

- Function based testing
- Class based testing
- Mark (skip, xfail)
- Parametrize
- Mocking

## Pre-requisites

```bash
pip install pytest
pip install requests
```

## Command

```bash
pytest <test_file>
```

To debug (or to run print statements), use `-s`

```bash
pytest -s <test_file>
```
